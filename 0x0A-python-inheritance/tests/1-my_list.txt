========================
 How to use 1-my_list.py
========================

This modual contain one class Mylist that is derivated from the class list.

MyList Methods
==============
print_sorted(self): Print the list in ascending sorted order


Usage
=====

::
	>>> MyList = __import__('1-my_list').MyList


::
	>>> my_list = MyList()
	>>> my_list.append(1)
	>>> my_list.append(4)
	>>> my_list.append(2)
	>>> my_list.append(3)
	>>> my_list.append(5)
	>>> print(my_list)
	[1, 4, 2, 3, 5]
	>>> my_list.print_sorted()
	[1, 2, 3, 4, 5]
	>>> print(my_list)
	[1, 4, 2, 3, 5]
